# GenBank Feature Table Processor v4.0

Convert GenBank files (.gb) to NCBI-compliant feature tables (.tbl) with automatic post-processing.

## Quick Start

Single file:
```bash
python genbank_processor_VD1.py input.gb
Batch processing:

bash
python genbank_processor_VD1.py -i input_dir/ -o output_dir/
```
