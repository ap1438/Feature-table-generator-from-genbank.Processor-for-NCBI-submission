# Contributing

Thank you for interest in contributing!

## How to Contribute

### Report Bugs
- Create an Issue
- Describe the problem
- Include example file if possible

### Suggest Features
- Create an Issue
- Describe the idea
- Explain the use case

### Submit Code
1. Fork the repository
2. Create a branch
3. Make changes
4. Submit a Pull Request

## Code Standards

- Follow PEP 8
- Add docstrings
- Include tests
- Update documentation

## Questions?

Create an Issue or Discussion!
